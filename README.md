# nexus-agent

NexusAI Multi-Agent System. Botan için çalışan otonom AI agentları.

## Agentlar

| Agent | Rol |
|-------|-----|
| 💻 Cortex | Yazılım geliştirme |
| 🛡️ Shield | Siber güvenlik |
| 🎬 Creator | İçerik üretimi |
| 💼 Freelance | İş geliştirme |
| 📚 Learn | Araştırma & öğrenme |
| 🚀 Deploy | DevOps |

## Kullanım

```bash
npm install
npm run dev
```
